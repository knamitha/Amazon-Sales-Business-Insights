# 📊 Amazon Sales Analytics Dashboard (Excel)

## 1. Overview

This project delivers an interactive **Amazon Sales Analytics Dashboard** built in Microsoft Excel. It translates transactional data into actionable insights across revenue, profitability, product performance, and regional distribution.

---

## 2. Problem Statement

Organizations require quick, reliable visibility into sales performance and profitability. This dashboard addresses that need by consolidating key metrics and enabling interactive exploration through filters.

---

## 3. Objectives

* Quantify **sales and profit performance** over time
* Identify **top/bottom products and categories**
* Analyze **regional distribution and customer locations**
* Enable **dynamic filtering** for decision-making

---

## 4. Dataset

* Source: Amazon Sales Dataset (Kaggle)
* Key fields: `Order Date`, `Sales`, `Profit`, `Category`, `Product Name`, `Customer Location`, `Order ID`

---

## 5. Tools & Techniques

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Slicers (interactive filtering)
* Conditional Formatting
* Data transformation (derived fields: Month, Year, Profit %)

---
